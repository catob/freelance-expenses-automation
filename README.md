# Freelance Expenses Automation

Google Apps Script project that automatically imports receipts from Gmail, saves them to Google Drive, and tracks expenses in a Google Sheet.

## Setup

Requires [clasp](https://github.com/google/clasp) (Google Apps Script CLI).

## Deploying

```bash
# Push local changes to Google Apps Script
clasp push

# Pull remote changes to local
clasp pull

# Open the Apps Script editor in browser
clasp open
```
